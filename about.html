About This Project

📌 Project Title

Cloud-Based Text to Image Generator for Education

🎯 Objective

The objective of this project is to develop a web-based application that generates educational images from text input using Artificial Intelligence. This helps students understand complex concepts visually.

⚙️ Technologies Used

• Frontend: HTML, CSS
• Backend: Python (Flask)
• Cloud API: Hugging Face (Stable Diffusion)

💡 Key Features

• Generate images from text input
• Subject-based image generation
• AI-enhanced prompts for better results
• Download generated images

📚 Educational Benefits

This system helps students and teachers by providing visual learning tools. It improves understanding of subjects like Biology, Physics, and Geography through AI-generated diagrams.
